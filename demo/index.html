makeshift.js

7 novel, interactive chart types that don't exist anywhere else. Zero dependencies. Pure SVG. Works everywhere.

v0.1.0 · MIT License · 0 dependencies

🌳 Decision Tree Builder

Build actual decision trees from data — not drawings. Automatic CART splitting, hover to see records, collapse subtrees.

📊 Gantt Chart

Dependency arrows that route around tasks. Critical path highlighting. Resource overload warnings. Milestone diamonds. Swimlanes.

💰 Cash Flow Waterfall

See where your money actually went. Click any block to see individual transactions. Running total line included.

🏢 Org Chart / Influence Map

Toggle between formal hierarchy and actual influence network. Node size = communication volume. Edge thickness = collaboration frequency.

🔥 Habit Heatmap

The GitHub contribution graph, but for anything. Streak counter. Goal scaling. Beautiful color gradients.

🔀 Before/After Slider

Drag the divider to reveal differences. Two complete charts on the same canvas, clipped at the split point.

📝 Word Diff Chart

Visualize how language changed over versions. Column view shows word flow with connectors. Cloud view shows the delta at a glance.
